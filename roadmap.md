# Roadmap — 8 тижнів

Принцип: кожен тиждень закриває один перевірний інкремент. Без зеленого тесту
тиждень не вважається закритим (`docs/ethics.md`, метод Insight-to-Artifact).

| Тиждень | Інкремент | Доказ закриття |
|---|---|---|
| 1 | MVP-ядро: 7 методів + контракт артефакту | `make test` зелений, `artifact_checker` чистий |
| 2 | 5 production-промптів + узгодження з методами | прогін кожного промпта на 3 кейсах |
| 3 | JSON-схеми + валідатори (intent/reflection/trace/artifact) | тести схем зелені |
| 4 | CLI `finalizer100` + API `/finalize`, `/intent` | smoke-тести API, CLI на прикладах |
| 5 | 10 прикладів використання з артефактами | `examples/` усі відтворювані |
| 6 | Інтеграційний шар з зовнішнім LLM (опційно, офлайн-замовчування) | контрактні тести адаптера |
| 7 | Документація: manifesto/philosophy/ethics/glossary + сайт-копія | review, без заборонених claims |
| 8 | Реліз v0.1.0: ліцензія, CI, публікація на neuron7xLab | теги, зелений CI, чистий mypy/ruff |

## Поза-скоупом (свідомо)
- AGI, симуляція свідомості, нейротерапія — назавжди поза скоупом.
- Будь-які наукові claims про мозок без peer-review.

## Метрика проєкту
Людина після використання методу краще формулює намір, бачить блокер, створює
артефакт і може перевірити результат.
