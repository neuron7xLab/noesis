# Глосарій

**Намір (intent)** — вектор бажаного результату, що передує дії. Відповідь на
«навіщо», не «що».

**Дзеркало (mirror)** — функція LLM повертати структуру наміру, а не істину про
світ. Протилежність оракулу.

**Рефлексія (reflection)** — дзеркальний опис власного мислення; сирий намір →
кристалізований.

**Інтроспекція (introspection)** — розклад нечіткого/емоційного запиту на намір,
блокер, обмеження, дію.

**Реверсивний інференс (reverse inference)** — мислення назад від цілі до першої
відсутньої умови.

**Екстрапольоване мислення (extrapolated thinking)** — розгортання рішення вздовж
часових горизонтів (now → 1d → 1w → 1m → 1y).

**Finalizer-100** — кристалізація контексту в параграф 90–110 слів з обов'язковими
якорями (намір, мета, блокер, дія, метрика, ризик).

**Артефакт (artifact)** — зовнішній перевірний об'єкт із 7 секцій контракту
(definition, input, method, output, validation, example, failure_modes).

**Екзокортекс (exocortex)** — LLM як зовнішнє розширення мислення (API, метапам'ять,
агенти), а не чат для відповідей.

**Сигнал vs шум** — намір (сигнал) проти емоцій/контексту/виправдань (шум).

**Режим відмови (failure mode)** — конкретний спосіб, у який метод чи поняття
ламається; обов'язкова секція кожного методу.
