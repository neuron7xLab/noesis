# GNWT_OPERATIONAL_ANALOGY

GNWT лише як ОПЕРАЦІЙНА аналогія (`noesis/broadcast.py`). Жодного claim про вимір
нейронного workspace / свідомість.

- **broadcast** — сигнал стає глобально доступним (стиснення-в-prompt → простір станів).
- **selective reentry** — людина/verifier обирає, що повертається в цикл.
- **workspace winner** — сигнал, що домінує поточний цикл (причинна категорія).
- **suppressed signal** — валідна, але не-broadcast інформація (критичний ризик).

Обов'язково ≥1 suppressed + оцінка useful/dangerous. Failure mode: гладкий вихід
перекриває намір; емоційна виразність б'є докази.
