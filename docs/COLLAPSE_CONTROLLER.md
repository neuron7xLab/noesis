# COLLAPSE_CONTROLLER

Коли спинити розширення й видати артефакт (`noesis/collapse_controller.py`).
Collapse у рівно один артефакт/дію, окрім human_review. Пояснює відкинуті кандидати,
зберігає невизначеність. Failure modes: передчасний collapse, нескінченне розширення,
хибна впевненість, втрата мінорного сигналу.
