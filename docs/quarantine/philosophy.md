# Філософія методу

## Базовий цикл
```text
слово → структура уваги → модель світу → рішення → дія → новий досвід → нове слово
```
Кожна ланка спостережувана й керована. Метод втручається у ланку «слово»,
бо вона найдешевша для зміни й має найбільший важіль на решту циклу.

## Дзеркало, а не оракул
LLM працює як дзеркало: повертає структуру наміру, а не істину про світ. Це
розрізнення критичне. Оракул претендує на знання; дзеркало повертає тобі тебе.
Усі методи спираються на дзеркальну, а не пророчу функцію.

## Інверсія як антидот самопідтвердженню
Reverse Inference і інверсна рефлексія навмисно перевертають хід думки: від цілі
назад, від протилежної гіпотези, від сліпої плями. Інверсія ламає інваріанти,
які людина не бачить зсередини власного потоку.

## Артефакт як умова існування думки
Думка, що не стала зовнішнім перевірним об'єктом, не існує для системи: її не
можна перевірити, передати, спростувати. Insight-to-Artifact робить
матеріалізацію обов'язковою.

## Межа застосування
Метод підсилює мислення людини; він не є джерелом істини, не є свідомістю і не
замінює фахівця (лікаря, психолога, юриста). Див. `ethics.md`.
