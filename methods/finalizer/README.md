# Finalizer Method — Стиснення хаосу до дії

> Великий контекст → 90–110 слів української, що містять усе потрібне для дії.

## definition
Кристалізація великого, розмитого контексту в один щільний параграф фіксованої
довжини, який обов'язково містить намір, мету, блокер, дію, метрику, ризик.

## input
Великий контекст: довга розмова, купа думок, сирий дамп свідомості.

## method
1. Подати контекст у LLM з промптом `prompts/finalizer_mirror.md`.
2. Отримати один параграф 90–110 слів українською.
3. Перевірити контракт через `tools/finalizer100.py` (CLI `finalizer100`).
4. Якщо `FAIL` — повернути на доопрацювання, не приймати.

## output
Параграф 90–110 слів з обов'язковими якорями: намір, мета, блокер, дія, метрика, ризик.

## validation
```bash
finalizer100 my_finalizer.md   # [OK] words=98 ... або [FAIL] + відсутні якорі
```
Або HTTP: `POST /finalize {"text": "..."}`.

## output_contract
- 90 ≤ слів ≤ 110.
- присутні всі шість сенсових якорів.

## example
Див. `examples/personal_decision/` — реальний фіналайзер, що проходить `[OK]`.

## failure_modes
- **Недотискання**: 70 слів, пів контексту втрачено → `FAIL` за довжиною.
- **Роздування**: 140 слів, повернувся шум → `FAIL`, стискай далі.
- **Втрата якоря**: немає метрики чи ризику → `FAIL`, контракт незакритий.
