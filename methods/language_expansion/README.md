# Language Expansion Method — Розширення мислення через мову

> Межі мови — межі доступного мислення. Точніше слово = ширша модель світу.

## definition
Розширення мисленнєвого простору навколо терміна через його дефініцію,
анти-дефініцію, сигнал, режим відмови та приклад використання.

## input
Один термін або поняття, яким ти оперуєш нечітко.

## method
Для терміна послідовно заповнити:
```text
term → definition → non-definition → signal → failure mode → use case
```
- **definition**: що це є (одне речення).
- **non-definition**: що це НЕ є (відсікти сусідні поняття).
- **signal**: за якою ознакою впізнати в потоці.
- **failure mode**: коли поняття ламається/не застосовне.
- **use case**: один конкретний приклад дії.

## output
Розширена картка терміна з п'ятьма гранями.

## validation
- Заповнені всі п'ять граней.
- `non-definition` справді відсікає, а не повторює `definition` іншими словами.

## example
```text
term:           "намір"
definition:     вектор бажаного результату, що передує дії
non-definition: не емоція, не план, не текст запиту
signal:         з'являється відповідь на питання "навіщо", а не "що"
failure mode:   зникає під тиском дедлайну — підміняється першою-ліпшою дією
use case:       перед задачею назвати намір вголос → відсіяти шумні підзадачі
```

## failure_modes
- **Псевдоглибина**: красиві абстракції без сигналу/прикладу → вимагай конкретику.
- **Синонімізація**: `non-definition` = перефразований `definition` → відсікай реально.
- **Нескінченне розгалуження**: одна картка на термін за раз.
