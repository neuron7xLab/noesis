# Security Policy

## Підтримувані версії
Підтримується остання мінорна гілка (`0.8.x`).

## Звітування про вразливість
Надсилай приватно на **neuron7x@ukr.net** із темою `SECURITY: noesis`. Не відкривай
публічний issue до узгодженого розкриття. Очікуй відповідь у межах 72 годин.

## Модель загроз — межі
Noesis — інструмент екстерналізованої метакогніції. Він **не**:
- виконує недовірений код із входу;
- є каналом для PII чи медичних/психіатричних даних (див. [`docs/ethics.md`](docs/ethics.md));
- гарантує приватність при використанні опціонального `[llm]` backend — у цьому
  режимі текст іде до зовнішнього API (Anthropic). Детермінований backend
  (default) працює повністю локально й нічого не надсилає.

## Залежності
Рантайм: `pydantic`, `fastapi`, `jsonschema`. Опціонально: `uvicorn` (`[serve]`),
`anthropic` (`[llm]`). Тримай їх оновленими; CI прогонить гейти на кожен PR.
