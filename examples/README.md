# Приклади використання (10 кейсів)

Чотири пророблені приклади з кодом/артефактами + шість сценаріїв застосування.

## Пророблені (з файлами)
1. **Особисте рішення** — `personal_decision/finalizer.md`
   Finalizer-100 перед вибором; проходить `finalizer100 → [OK]`.
2. **Дослідницьке мислення** — `research_thinking/reverse_inference.md`
   Reverse Inference знаходить реальну першу відсутню умову (відтворюваність).
3. **Дизайн продукту** — `product_design/insight_to_artifact.md`
   Інсайт → артефакт із 7 секцій, перевірений `artifact_checker`.
4. **Протокол навчання** — `learning_protocol/language_expansion.md`
   Розширення мови навколо терміна як цикл засвоєння.

## Сценарії (метод → ситуація)
5. **Reflection** — розплутати ранкову тривогу «все валиться» в одну дію.
6. **Introspection** — відокремити справжній намір зміни роботи від втоми.
7. **Extrapolated Thinking** — прорахувати наслідки публічного релізу сирої ідеї.
8. **Finalizer** — стиснути 2-годинну розмову з командою в один абзац рішення.
9. **Reverse Inference** — від «хочу запустити продукт» до першого відсутнього кроку.
10. **Language Expansion** — зняти плутанину між «намір» і «план» у формулюванні задач.

> Кожен пророблений кейс можна перевірити: `make test` і `finalizer100 examples/personal_decision/finalizer.md`.
